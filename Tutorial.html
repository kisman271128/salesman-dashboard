🕐 Setup Task Scheduler - Morning Update Otomatis

📍 Lokasi File: C:\Dashboard\morning_update.bat
⏰ Target Waktu: Setiap hari jam 07:00 pagi
🎯 Tujuan: Update dashboard otomatis tanpa manual intervention

📋 Step 1: Buka Task Scheduler

🚀 Cara Membuka:
  1. Tekan Windows Key + R
  2. Ketik: taskschd.msc
  3. Tekan Enter
💡 Alternative: Start Menu → ketik "Task Scheduler" → Enter

📋 Step 2: Create Basic Task

➕ Di Task Scheduler window:
  1. Klik "Create Basic Task..." di panel kanan
  2. Atau klik menu Action → Create Basic Task

📋 Step 3: Task Information

📝 Isi form berikut:
Field Isi dengan
Name Morning Dashboard Update
Description Update salesman dashboard setiap pagi jam 7:00

Klik "Next"

📋 Step 4: Task Trigger (Schedule)

⏰ Pilih trigger:
  1. Pilih "Daily"
  2. Klik "Next"

📋 Step 5: Daily Settings

📅 Konfigurasi schedule:
Setting Value
Start Tanggal hari ini
Time 07:00:00 (7:00 AM)
Recur every 1 days

Klik "Next"

📋 Step 6: Action (Program to Run)

⚙️ Pilih action:
  1. Pilih "Start a program"
  2. Klik "Next"

📋 Step 7: Program Settings

📂 Konfigurasi program:
Field Value
Program/script C:\Dashboard\morning_update.bat
Add arguments (kosongkan)
Start in C:\Dashboard
⚠️ Penting:
  • Pastikan path C:\Dashboard\morning_update.bat benar-benar ada
  • Field "Start in" harus diisi C:\Dashboard

Klik "Next"

📋 Step 8: Summary & Finish

✅ Review settings:
  1. Review semua setting yang sudah diisi
  2. Centang "Open the Properties dialog for this task when I click Finish"
  3. Klik "Finish"

📋 Step 9: Advanced Properties (Optional)

🔧 Di Properties dialog:

General Tab:

  • ✅ "Run whether user is logged on or not"
  • ✅ "Run with highest privileges"
  • Configure for: "Windows 10" atau "Windows 11"

Conditions Tab:

  • ✅ "Wake the computer to run this task"
  • ❌ "Start the task only if the computer is on AC power" (uncheck)

Settings Tab:

  • ✅ "Allow task to be run on demand"
  • ✅ "Run task as soon as possible after a scheduled start is missed"
  • Stop task if runs longer than: "1 hour"

Klik "OK" untuk save

🧪 Testing & Verification

Test Manual Run:

🔍
  1. Di Task Scheduler, cari task "Morning Dashboard Update"
  2. Right-click → "Run"
  3. Monitor command window untuk melihat progress
  4. Check jika berhasil tanpa error

Check Task History:

📊
  1. Select task "Morning Dashboard Update"
  2. Klik tab "History" di bawah
  3. Jika tidak ada, enable via View → Show History
  4. Monitor untuk successful runs
✅ Task berhasil dibuat!
Sekarang dashboard akan update otomatis setiap pagi jam 7:00

Task Tidak Jalan:

  • Verify path file: C:\Dashboard\morning_update.bat
  • Run task manual untuk test
  • Check Windows Event Viewer untuk error details

Task Jalan tapi Gagal:

  • Check file morning_update.log di C:\Dashboard\
  • Pastikan Excel file sudah di-update dan di-save
  • Verify internet connection untuk GitHub push
  • Test Python script manual: python morning_update.py

🚀 Quick Commands

Manual Run (Backup):

cd C:\Dashboard morning_update.bat

Check Task Status:

schtasks /query /tn "Morning Dashboard Update"

Disable Task (Temporary):

schtasks /change /tn "Morning Dashboard Update" /disable

Enable Task:

schtasks /change /tn "Morning Dashboard Update" /enable
